# 浏览器存储

为了减轻后端 API 的负载并提高性能，PRUNplanner 直接在你的浏览器中存储游戏数据。这可以实现更快的访问速度和更顺畅的用户体验。该工具还会在设定时间后自动刷新存储的数据，以确保其保持最新状态。

## 存储类别

- **游戏数据 (Game Data)**
  这主要包括静态信息，如物料、建筑、配方、星球数据和交易所详情。它源自 FIO，或者在交易所数据的情况下，由 PRUNplanner 计算得出。交易所数据的刷新频率更高，以确保利润和成本计算的准确性。

- **规划数据 (Planning Data)**
  此类包含你的个人规划信息，包括你的帝国、计划、交易所偏好以及任何分享的计划。

- **FIO 数据 (FIO Data)**
  这指的是你的游戏内个人数据，如仓库和站点信息。要访问此类数据，你可以通过提供用户名和 FIO API 密钥来关联你的 Prosperous Universe 账户。PRUNplanner 将定期检查更新，并自动从后端同步最新数据。

# 偏好设置

偏好设置允许你保存常用的设置和配置，使你的规划过程更高效且更具个性化。

例如：

- **默认帝国**
  当你访问帝国视图时，会自动加载你首选的帝国，从而无需每次手动选择。

- **消耗视图阈值 (Burn View Thresholds)**
  如果你将 PRUNplanner 与 FIO 结合使用来监控基地库存，可以为库存水平设置红色和黄色阈值。这些阈值可帮助你确定何时派遣补给船，并保存在你的浏览器中以便使用。

工具范围内的偏好设置可以直接从你的个人资料中管理，或者在相关视图中进行调整。计划特定设置可以在每个关联的计划中修改。
